### Added

- Initial release: MingBo Seal Server — enterprise e-seal & approval workflow backend
- Seal lifecycle management (create, bind, authorize, revoke, destroy)
- Flowable BPMN 2.0 workflow engine integration
- Visual process designer support (countersign, or-sign, conditional branches)
- Form template management with dynamic form rendering
- Role-based access control (menu, button, data three-tier permission)
- JWT stateless authentication with multi-terminal support
- Swagger/SpringDoc auto-generated API documentation
- Server monitoring (CPU, memory, disk, JVM)
- Redis cache monitoring and statistics
- Druid connection pool monitoring and slow SQL detection
- Online user session management
- Scheduled task management (Quartz)
- Code generator (Velocity-based CRUD scaffolding)
- Bilingual documentation (English / 简体中文) with screenshots
